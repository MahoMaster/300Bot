# 命令文档

`命令 M参数 S参数`

命令+空格+参数前缀+参数+空格+参数前缀+参数

## 基础

| 命令 | 参数 | 示例 | 说明 |
| ----- | ------- | ------- | ---- |
| `帮助`/`使用说明`/`help` | - | `help` | 说明文档 |
| `涩图`/`来张涩图`/`整点二次元` | - | `整点二次元` | 获得一张二次元图片 |
| `点歌` | `歌曲关键信息` | `点歌 天体观测 爱美` | 分享搜索到的歌曲 |
| `天气`/`查天气`/`天气预报`/`当前天气` | `地区名称` | `当前天气 成都` | 获取实时天气状况 |
| `送礼`/`免费礼物` | `艾特成员/成员qq` | `送礼 675559614`/`免费礼物 @Maho` | 机器人赠送礼物 |
